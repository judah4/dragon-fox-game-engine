# Untitle Dragon Game Engine Test Game

This is for testing the game! So far there is just a spinning square that textures can be loaded from disk to.

# Controls

## Keyboard

WASD - Move
QE - Look left and right
Space - Up
Left Control - Down
Arrow Keys - Look
T - Load textures
Right Click + Mouse - Look

## Controller

Left Stick - Move
Right Stick - Look
Right Trigger - Up
Left Trigger - Down
